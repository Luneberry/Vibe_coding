# 🧘 Project7: Daily Reflection App

매일의 생각을 기록하고 돌아볼 수 있는 채팅 형태의 웹 애플리케이션입니다.

---

## 실행 방법

```bash
# 폴더 이동
cd daily-reflection-app

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

## 주요 기술

*   Next.js
*   TypeScript
*   Tailwind CSS

## 환경 변수

실행하기 전에 `.env.local` 파일을 생성하고 필요한 환경 변수를 설정해야 합니다. `.env.local.example` 파일을 참고하세요.
