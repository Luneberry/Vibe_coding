간단한 로직에 따라 비트코인 거래를 자동화하는 Python 스크립트입니다.

### 실행 방법

```bash
# 필요한 라이브러리가 있다면 설치해주세요 (e.g., pip install requests)
python BTC-automation.py
```

---

## 2. coin-chart-app 📊

Next.js를 사용하여 코인 시세 데이터를 차트로 보여주는 웹 애플리케이션입니다.

### 실행 방법

```bash
# 폴더 이동
cd coin-chart-app

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

### 주요 기술

*   Next.js
*   TypeScript
